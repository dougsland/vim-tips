# vim-tips
Just another notes from VIM

## Set Tab
Sets a tab to be four spaces
```
:set tabstop=4
```

## Autoindent
```
:set autoindent
```

## Do not highlight searched words
```
:set nohlsearch
```

## Show when insert mode is set
```
:set showmode
```
## To paste code
### Set paste mode
```
:set paste
```

### Disable paste mode:
```
:set nopaste
```

### Set a keyboard key (F2) to pastemode
```
$ ~/.vimrc
set pastetoggle=<F2>
```
