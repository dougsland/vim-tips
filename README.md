# vim-tips
Just another notes from VIM

## To paste code
### Set paste mode
```
:set paste
```

### Disable paste mode:
```
:set nopaste
```

### Set a keyboard key to pastemode
```
$ ~/.vimrc
set pastetoggle=<F3>
```
